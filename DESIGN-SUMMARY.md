# Design summary

- Design concept: Institutional bluebook
- Layout personality: traditional institutional
- Typography: Libre Baskerville headings with Work Sans body text
- Color palette: primary #1e3a8a, secondary #93c5fd, accent #b45309, background #eff6ff, text #172554, muted #64748b
- Shape language: Slight rounding
- Visual motif: rule lines and numbered briefs
- Unique layout choices: first sections use Selection dossier, then a custom service index, process ribbon, question block, and resource strip with varied widths and alignments.
- Difference from previous sites: this redesign changes the hero composition, section ordering, image treatment, card style, spacing rhythm, button shape, and footer treatment for this domain.
